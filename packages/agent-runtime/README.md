# Agent Runtime

This package will coordinate GraphCode's AI-assisted workflows.

Planned responsibilities:

- Run local-agent workflows scoped to selected nodes or subgraphs.
- Return reviewable diffs instead of silently mutating source files.
- Attach test output, blast-radius hints, and rationale to proposal cards.
- Run global-agent workflows for architecture summaries, ripple-effect detection, and follow-up task suggestions.
- Keep direct whole-repository rewrites out of the initial demo path.

No agent runtime implementation has been added yet.
