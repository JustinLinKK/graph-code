# Local Server

This directory will hold the local service that connects the web workspace to the user's repository.

Planned responsibilities:

- Scan and index target repositories.
- Refresh graph snapshots after file changes.
- Read source files and resolve node source ranges.
- Run available tests, linters, or checks for proposed edits.
- Expose APIs for graph data, file access, and reviewable diff proposals.

No server implementation has been added yet.
